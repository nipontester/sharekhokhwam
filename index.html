กำลังเชื่อมต่อ…

อีกนิดเดียว — ตั้งค่า Supabase ก่อน

  1. สร้างโปรเจกต์ที่ supabase.com แล้วเปิด SQL Editor รันไฟล์ supabase/schema.sql
  2. ไปที่ Project Settings → API คัดลอก Project URL และ anon key
  3. วางค่าทั้งสองลงใน js/config.js แล้ว deploy ใหม่ / รีเฟรชหน้านี้

คีย์ anon ออกแบบมาให้อยู่ฝั่งเบราว์เซอร์ได้ — สิทธิ์การอ่าน/เขียนควบคุมด้วย Row Level Security
